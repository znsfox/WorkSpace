### 汇编语言中的"#"号是什么意思？

```
立即数寻址
比如：
MOV AX, #30H
执行后AX中的内容是30H
而
MOV AX, 30H
执行后AX中的内容是30H这个内存地址里保存的数值
```

汇编语言中的”[]"是什么意思？

```
SI 是寄存器寻址
[SI]是寄存器间接寻址
[SI+2]是相对的寄存器间接寻址，又称直接变址寻址，它也可以写成2[SI]这样的形式
[BX+SI]是基址变址寻址，也可写成[BX][SI]
```
